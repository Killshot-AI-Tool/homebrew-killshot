# homebrew-killshot

Homebrew tap for the [killshot](https://killshot.io) CLI.

## Install

```bash
brew tap Killshot-AI-Tool/killshot
brew install killshot
```

Or as a single command:

```bash
brew install Killshot-AI-Tool/killshot/killshot
```

## How this tap is maintained

`Formula/killshot.rb` is **auto-written by [goreleaser](https://goreleaser.com)**
from [Killshot-AI-Tool/killshot](https://github.com/Killshot-AI-Tool/killshot)
on every release. Do not hand-edit — your edits will be overwritten the next
time a tag is pushed.

Release cadence: tag-driven from the source repo (`git tag vX.Y.Z`).
